Jeu du nombre aleatoire (Python)


Petit jeu en Python réalisé pour s'entraîner à programmer.

Principe du jeu


L'ordinateur choisit un nombre aléatoire et le joueur doit le deviner.

Après chaque tentative, le programme indique si le nombre est :


- trop grand

- trop petit

Le jeu continue jusqu'à ce que le joueur trouve le bon nombre.

Niveaux de difficulté


Le joueur peut choisir entre trois niveaux :


1. Facile : nombre entre 1 et 50

2. Moyen : nombre entre 1 et 100

3. Difficile : nombre entre 1 et 1000

Fonctionnalités

- génération de nombre aléatoire

- choix de la difficulté

- compteur de tentatives

- vérification des entrées utilisateur

- gestion des erreurs si l'utilisateur n'entre pas un nombre

Technologies utilisées

- Python

Lancer le jeu

1. Télécharger le fichier Python

2. Ouvrir un terminal dans le dossier

3. Lancer la commande :
	python nom_du_fichier.py

Objectif du projet


Ce projet a été réalisé pour apprendre :


- les boucles (while)

- les conditions (if / elif / else)

- la gestion des erreurs (try / except)

- l'utilisation du module random

Projet réalisé dans le cadre d'un apprentissage de Python.
